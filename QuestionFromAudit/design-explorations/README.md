# Trust Instance Page — Design Explorations

Three alternative design directions for the trust instance page (`/instance/[trustLinkId]`).

Open any HTML file directly in a browser to preview.

**Figma file:** https://www.figma.com/design/WL7ugItfjIeKFx6STuZvvN

## Designs

### V1: Clean Clinical (`v1-clean-clinical.html`)
Minimal card-based redesign. Generous whitespace, left color accents per category, inline answer chips, floating edit panel with vertical icon sidebar.
- Figma node: `7:2`

### V2: Notion-style (`v2-notion-style.html`)
Sidebar + content pane layout. Collapsible category groups, full document-style main area, inline editing on hover, "Review Changes" FAB.
- Figma node: `9:2`

### V3: Dashboard Grid (`v3-dashboard-grid.html`)
Responsive tile grid with slide-in detail drawer. Progress bar, filter chips, pinnable drawer, hover quick-actions.
- Figma node: `11:2`
